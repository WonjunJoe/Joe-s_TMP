Section 02

Claude Code Skills

SKILL.md 파일로 정의하며, Claude가 상황에 맞게 자동 활성화하는 확장 시스템입니다. Superpowers 프레임워크가 가장 널리 사용됩니다.

01

Test-Driven Development

테스트 먼저, 구현은 나중에. Red-Green-Refactor 사이클을 강제하여 안정적인 코드를 보장합니다.

방법론
02

Systematic Debugging

버그 발견 시 체계적으로 원인을 추적하는 프로세스. 추측 대신 증거 기반 디버깅을 수행합니다.

디버깅
03

Brainstorming

구현 전 사용자 의도, 요구사항, 설계를 탐색합니다. 모든 창의적 작업의 시작점입니다.

설계
04

Code Review

코드 리뷰 피드백을 기술적 엄밀성으로 검증합니다. 맹목적 동의 대신 기술적 판단을 우선합니다.

품질
05

Writing Plans

멀티스텝 작업의 구현 계획을 체계적으로 수립합니다. 코드 전에 아키텍처를 먼저 설계합니다.

설계
06

Git Worktrees

격리된 Git worktree에서 안전하게 실험적 작업을 수행합니다. 메인 브랜치를 보호합니다.

인프라
07

Frontend Design

AI 슬랍이 아닌 진짜 디자인. 대담한 미학적 방향성과 프로덕션급 UI를 생성합니다.

디자인공식
08

Parallel Agents

독립적인 2개+ 작업을 병렬 서브에이전트로 동시 처리합니다. 효율성을 극대화합니다.
