1. AI Agent 개괄

AI Agent는 LLM이 도구를 사용하고, 판단하고, 반복적으로 행동하며 복잡한 작업을 자율적으로 수행하는 시스템입니다. 2026년 현재, 조직의 57%가 프로덕션에서 AI 에이전트를 운영 중입니다.

핵심 개념

  • 에이전트 루프 — LLM이 관찰 → 판단 → 도구 호출 → 결과 해석을 반복하는 사이클
  • 도구 사용 (Tool Use) — 파일 편집, 셸 명령, API 호출 등 외부 시스템과 상호작용
  • 서브에이전트 — 복잡한 작업을 병렬로 처리하기 위해 하위 에이전트를 생성
  • MCP (Model Context Protocol) — AI 모델과 외부 도구를 연결하는 표준 프로토콜

Claude Agent SDK

Claude Code의 에이전트 기능을 프로그래밍으로 활용

Python(claude-agent-sdk-python)과 TypeScript(claude-agent-sdk-typescript) 두 가지 버전을 제공합니다. Claude Code와 동일한 에이전트 루프, 도구, 컨텍스트 관리를 프로그래밍 방식으로 사용할 수 있습니다.

주요 기능: 파일 작업, bash 명령, 웹 검색, MCP 통합, Computer Use

3. Plugins & MCP 서버

MCP(Model Context Protocol)는 Anthropic이 공개한 오픈 프로토콜로, AI 모델과 외부 도구/데이터를 연결하는 표준입니다. 현재 1,200개+ MCP 서버가 존재하며, OpenAI와 Google도 채택했습니다.

필수 MCP 서버

GitHub MCP Server

이슈, PR, 저장소 관리를 Claude에서 직접 수행합니다.

공식필수

Context7

라이브러리/프레임워크의 최신 공식 문서를 실시간으로 가져옵니다.

문서필수

Filesystem

설정 가능한 접근 제어를 통한 안전한 파일 시스템 작업을 제공합니다.

공식 레퍼런스

Memory

지식 그래프 기반의 영속적 메모리 시스템입니다.

공식 레퍼런스

Sequential Thinking

동적이고 반성적인 사고 시퀀스를 통해 복잡한 문제를 해결합니다.

추론

Slack / Notion / Zapier

협업 도구와 연동하여 워크플로우를 자동화합니다.

통합

MCP 서버 찾기

  • modelcontextprotocol/servers — 공식 레퍼런스 서버 저장소
  • awesome-mcp-servers — 커뮤니티 큐레이션 목록

Claude Code Plugins 시스템

2025년 10월 퍼블릭 베타로 출시된 플러그인 시스템은 슬래시 커맨드, 에이전트, 훅, MCP 서버를 확장합니다.

  • Hooks — 파일 저장, 커밋 등 이벤트에 자동 트리거되는 스크립트
  • Slash Commands — 커스텀 명령어로 반복 작업을 자동화
  • 서브에이전트 — 병렬 작업 디스패치로 복잡한 작업을 분할 처리

4. OpenClaw 프로젝트

로컬에서 실행하는 개인 AI 어시스턴트 — 33만+ Stars

20개 이상의 메시징 플랫폼을 하나로 통합하는 오픈소스 프로젝트입니다.

주요 특징

  • 멀티 채널 통합 — WhatsApp, Telegram, Slack, Discord, Signal, iMessage, Teams, Matrix 등 20개+
  • 로컬 우선 아키텍처 — 데이터가 사용자 기기에서 처리됨
  • 음성 기능 — Wake and Talk 모드 (ElevenLabs 연동)
  • Live Canvas — 에이전트 주도 시각적 워크스페이스
  • ClawHub — 확장 가능한 스킬 레지스트리
  • 브라우저 자동화 — Chrome/Chromium 제어

기술 스택

항목내용
언어TypeScript/JavaScript (Node.js 24 권장)
구조모노레포 (apps, packages, skills, extensions)
메시징Baileys (WhatsApp), grammY (Telegram), Bolt (Slack), discord.js
UIReact 기반 Control UI + WebChat
배포npm, Docker, Nix, systemd/launchd

빠른 시작

npm install -g openclaw@latest
openclaw onboard --install-daemon
